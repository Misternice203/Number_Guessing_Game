### Number_Guessing_Game

Simple CLI number guessing game using python.

## Code
[NumberGuessing.py](https://github.com/user-attachments/files/25821604/NumberGuessing.py)

## Features

- Storing and updating variables
- Importing modules
- Generated random decimals and integers
- Improved message formatting for better readability

## Technologies

- Python random module
- Command line interface
- if / elif / else logic
- Loops
- Random number generator

## How to run program

1. Clone repository
2. Navigate to the project folder
3. Run program using NumberGuessing.py file

## Example

- Guess the number between 1 and 5
- Enter your guess: 2
  Too low! Try again.
- Enter your guess: 4
  Too high! Try again.
- Enter your guess: 3
  Correct you guessed the number!
  Would you like to play again? (y/n):
